Étude sur l’utilisation d’Internet chez les étudiants
Cette étude a examiné l'impact de l'utilisation d'Internet sur les résultats scolaires des étudiants. Les données ont été recueillies au moyen d'un sondage auprès de plusieurs étudiants afin de déterminer le nombre d'heures passées en ligne quotidiennement. Les résultats ont montré que la plupart des étudiants passent entre 3 et 5 heures par jour en ligne, principalement sur les réseaux sociaux et pour étudier. Le logiciel Excel avancé a été utilisé pour organiser et analyser ces données à l'aide de graphiques et de tableaux détaillés. En conclusion, l'étude souligne l'importance d'une utilisation équilibrée d'Internet pour éviter des effets négatifs sur la réussite scolaire.
